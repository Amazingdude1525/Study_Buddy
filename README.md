**AI Vityarthi** is a premium, high-performance study companion designed to elevate your focus and learning efficiency. Built with a cutting-edge glassmorphic UI and powered by advanced AI models for sentiment analysis, resource curation, and study planning.

## ✨ Features
- 🧠 **AI Study Advisor**: Neural study patterns and personalized guidance.
- 🎭 **Face Mood Detection**: Live sentiment analysis to adapt your study environment.
- 🎵 **Smart Lofi Streaming**: Mood-synced music integration.
- 💻 **Multi-Language Sandbox**: Compile Python, C, and C++ in a unified terminal.
- 📊 **Contribution Heatmap**: Visual tracking of your study progress.
- 🎯 **Goal Management**: Track and achieve your academic milestones.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend**: FastAPI (Python 3.12), SQLModel, Google OAuth 2.0.
- **AI/ML**: DeepFace, TensorFlow, Gemini AI.

## 📦 Local Setup

### 1. Backend (FastAPI)
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```

### 2. Frontend (React)
```bash
npm install
npm run dev
```
